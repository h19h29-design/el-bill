# Calculation Notes

## 데이터 하네스

- 기본 샘플은 첨부 엑셀의 2020~2025 학년도별 월분, 총 전기요금, 사용량을 익명화해 구성했다.
- 실제 학교명은 `A고등학교`로 대체했다.
- 고객번호, 담당자명, 연락처, 주소는 샘플에서 마스킹했다.
- 원본 파일은 repo에 저장하지 않는다.

## 엑셀 파서

자동 병합 규칙:

1. 시트명에서 `20`, `21`, `22`, `23`, `24`, `25` 같은 연도를 찾는다.
2. `월분` 헤더가 있는 행을 찾는다.
3. 해당 학년도 열, `사용량(kwh)` 열을 찾는다.
4. 월, 사용량, 총 전기요금이 숫자로 확인되는 행만 `MonthlyBill`로 변환한다.
5. 실패하면 업로드 화면의 수동 컬럼 매핑을 사용한다.

필수 매핑은 연도, 월, 사용량, 총 전기요금이다. 요금적용전력, 최대수요전력, 기본요금, 전력량요금 등은 선택 매핑이며 누락 시 추정값을 쓴다.

## 요금제 비교

요금 단가는 `src/data/ratePlans.ts`에 관리하며 UI 설정에서 수정 가능하다.

기본 고압A 단가:

| 구분 | 선택요금 I | 선택요금 II |
| --- | ---: | ---: |
| 기본요금 | 5,550원/kW | 6,370원/kW |
| 봄·가을철 | 86.5원/kWh | 82.1원/kWh |
| 여름철 | 123.3원/kWh | 118.8원/kWh |
| 겨울철 | 109.3원/kWh | 104.8원/kWh |

자동진단 후보 선정:

1. 학교 프로필의 계약종별, 수전전압, 현재 요금제명으로 현재 요금제를 찾는다.
2. 전체 요금표에서 `교육용` 계약종별 후보를 필터링한다.
3. 같은 계약종별/수전전압 후보를 우선 비교하고, 다른 학교용 요금제는 추가 검토 후보로 분류한다.
4. 최근 12개월 절감액을 기준으로 후보를 정렬하고 TOP 3를 결과 표에 표시한다.

계산 모드 A: 고지서 기반 차액 추정

- 기존 실제 총 청구액을 기준으로 삼는다.
- 현재 요금제와 후보 요금제의 기본요금 차액, 전력량요금 차액을 중심으로 절감액을 계산한다.
- 부가세, 전력산업기반기금, 기후환경요금, 연료비조정액 등은 기존 고지서의 기본요금+전력량요금 대비 비율로 보정한다.
- 실제 청구서 금액을 기준으로 하므로 공모전 MVP의 기본 자동진단 모드로 사용한다.

계산 모드 B: 요금표 기반 전체 추정

- 월 사용량, 요금적용전력, 계절별 단가표를 사용해 현재안과 후보안을 모두 다시 추정한다.
- 설정 화면에서 단가표와 보정계수를 수정할 수 있도록 유지한다.
- 단가표가 최신 한전 고시와 다를 수 있으므로 공식 청구액 계산이 아니라 내부 진단용 비교로만 사용한다.

공통 계산 흐름:

1. 월 사용시간 = 월 사용량 kWh / 요금적용전력 kW
2. 계절별 전력량 단가를 적용
3. 예상 최대수요전력 입력값이 있으면 기본요금 산정 전력에 반영
4. 전체/여름/겨울 사용량 증가율을 반영
5. 기후환경요금, 연료비조정액, 부가세, 전력산업기반기금은 진단용 추정 계수로 반영
6. 최근 12개월과 최근 3년 기준으로 현재안과 변경안을 비교

계산 신뢰도:

- `데이터 충분`: 최근 36개월 이상 자료가 있고 최대수요전력 컬럼이 모두 확인됨
- `보통`: 최근 12개월 이상 자료가 확인됨
- `낮음`: 12개월 미만이거나 필수 컬럼이 부족함

판정:

- 12개월 데이터가 없으면 `추가 검토 필요`
- 최근 12개월과 최근 3년 모두 절감이고 절감률이 3% 이상이면 `변경 추천`
- 변경 시 증가하면 `유지 추천`
- 나머지는 `추가 검토 필요`

## 피크관리

위험도:

- 80% 미만: 안전
- 80% 이상 90% 미만: 주의
- 90% 이상 95% 미만: 경고
- 95% 이상: 위험

검증 기준인 목표 500kW, 예상 485kW는 97%로 계산되며 `위험`으로 표시된다.

## 한전 파워플래너 자료

앱은 한전 파워플래너를 대체하지 않고, 학교 행정실 업무에 맞게 파워플래너/한전고지서 자료를 재분석하는 보조 플랫폼이다.

MVP 원칙:

- 한전 계정 자동 로그인 금지
- 크롤링 금지
- 비공식 API 호출 금지
- 사용자가 파워플래너에서 내려받거나 정리한 엑셀/CSV만 업로드
- 시연용 데이터는 기존 정책대로 24시간 후 자동 삭제

업로드 가능한 데이터 유형:

1. 월별 사용량
2. 일별 사용량
3. 시간대별 사용량
4. 최대수요전력
5. 예상요금
6. 소비패턴 분석 결과

파워플래너 자료가 있으면 피크관리 화면에서 시간대별 사용량 그래프와 최대부하 시간대 위험도를 추가로 표시한다. 자료가 없으면 기존 한전 고지서 월별 데이터와 예상 피크 입력값만으로 진단한다.

파워플래너의 `월별청구요금.xls`는 실제 바이너리 엑셀이 아니라 HTML 표를 `.xls` 확장자로 내려주는 형식일 수 있다. 공통 업로드 파서는 이 형식을 감지해 `연월`, `계약전력(kW)`, `요금적용전력(kW)`, `사용전력량(kWh)`, `사용일수(일)`, `지상역률(%)`, `진상역률(%)`, `청구요금(원)` 헤더와 행을 복원한다. 고지서 입력 화면에서는 월별 청구요금 행으로 자동 변환하고, 파워플래너 자료 가져오기 화면에서는 `연월` 기반 월별 사용량으로 자동 매핑한다.

향후 공식 API 또는 기관 연계가 가능할 경우를 대비해 `DataSource<TRecord>`와 `PowerPlannerDataSource` 타입을 분리했다. 현재 구현은 `provider: 'kepco-power-planner'`인 사용자 업로드 출처만 사용한다.

## 문서 생성

첨부 계획서/PDF 공문에서 확인한 구조를 바탕으로 다음 문서를 생성한다.

- 내부 결재용 변경 계획(안)
- 한전 제출 공문
- 전기사용계약 변경신청서 미리보기
- 붙임 체크리스트
- 계산 근거 요약표
- 담당자 검토 필요 항목 목록

모든 문서에는 내부 진단용 추정, 요금제 변경 신청은 원칙적으로 1년에 한 번만 가능하므로 예상 절감액과 피크 시나리오를 신중히 검토해야 한다는 문구, 실제 제출 전 담당자 검토 필요 문구를 포함한다.
